Most important Linux commands

ls - list выводит содержимое коталога

mv - move перемещение файла и директори. С помощью этой комманды можно переименовывать файлы исли вместо пути указать новое имя.

PWD - Print Working Directory выводит полное имя директории в который вы находитесь.

mkdir - make directory - создает новую директорию.

history - выводит список использованных в терминале комманд.

cd - Change Directory меняет рабочую директорию

copy - копирует файл

man - manual вызывает справочник комманд
